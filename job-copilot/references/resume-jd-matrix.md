## 1. ATS Keyword Extraction Logic

### Tier 1 — Must Include (exact match required)
- Job title stated in posting
- Required technical skills listed explicitly
- Certifications named as required
- Tools or platforms listed as "must have"

### Tier 2 — Should Include (synonyms acceptable)
- Preferred skills / "nice to have" items
- Methodologies mentioned (Agile, Scrum, OKRs, JTBD, etc.)
- Industry terms (fintech, SaaS, B2B, healthcare, etc.)
- Soft skills that appear more than once in JD

### Tier 3 — Context Keywords (weave into bullets naturally)
- Company mission language ("scale", "growth", "impact")
- Team structure hints ("cross-functional", "stakeholders")
- Output types ("roadmaps", "dashboards", "pipelines", "models")

### Keyword Placement Priority
```
1. Professional Summary (2–3 Tier 1 keywords)
2. Skills section (all Tier 1 + top Tier 2)
3. First bullet of most recent role (1–2 Tier 1)
4. Remaining bullets (Tier 2 + Tier 3 distributed)
```

## 2. Bullet Formula Library

### CAR Formula (default)
```
[Action verb] + [what you did] + [result with metric]

Example:
"Reduced API response time by 40% by refactoring authentication middleware and 
implementing Redis caching, cutting peak server load from 8s to 4.7s."
```

### STAR-Lite Formula (for complex situations)
```
[Context in 5 words or fewer] → [Action] → [Result]

Example:
"Inherited failing product launch — rebuilt roadmap with 3 cross-functional teams,
shipping MVP 2 weeks ahead of schedule with 94% stakeholder approval."
```

### Scale Formula (for ops / growth / data roles)
```
[Action] + [what] + [at X scale] + [outcome]

Example:
"Automated ETL pipeline for 12M daily records using Apache Airflow, reducing 
manual data prep from 6hrs to 18min per day."
```

### Leadership Formula (for senior / management roles)
```
[Led/Built/Scaled] + [team/initiative size] + [to achieve X outcome]

Example:
"Built and mentored a 7-person data science team from scratch, increasing model 
deployment velocity by 3x over 18 months."
```

## 3. Section Templates by Role Type

### Software Engineer / Developer
```
SKILLS SECTION:
Languages: [primary langs] | [secondary langs]
Frameworks: [backend] | [frontend]
Cloud & Infra: [platforms] | [tools]
Databases: [relational] | [NoSQL]
Practices: [Agile/TDD/CI-CD/etc]

EXPERIENCE BULLETS FOCUS:
- Features shipped (user impact, adoption %)
- Performance improvements (latency, throughput)
- Scale handled (requests/day, data volume)
- Code quality (coverage %, bugs reduced)
- Team collaboration (PR reviews, mentorship)
```

### Data Scientist / ML Engineer
```
SKILLS SECTION:
Languages: Python, R, SQL, [others]
ML/DL: [frameworks — sklearn, TF, PyTorch, etc.]
Data Stack: [Spark, Airflow, dbt, BigQuery, etc.]
Visualisation: [Tableau, PowerBI, Matplotlib, etc.]
Cloud: [AWS/GCP/Azure ML services]

EXPERIENCE BULLETS FOCUS:
- Model accuracy / lift over baseline
- Business impact ($, %, decisions influenced)
- Data scale (rows, features, pipelines)
- Deployment and monitoring in production
- Cross-functional stakeholders served
```

### Product Manager
```
SKILLS SECTION:
Product Tools: Jira, Confluence, Figma, Amplitude, Mixpanel
Methodologies: Agile, OKRs, Jobs-to-be-Done, User Story Mapping
Research: User Interviews, A/B Testing, Usability Studies

EXPERIENCE BULLETS FOCUS:
- Products shipped (DAU, MAU, revenue impact)
- Roadmap ownership (number of features, sprint velocity)
- Stakeholder alignment (exec, eng, design, sales)
- Metrics moved (retention, conversion, NPS)
- Team size influenced / led
```

### Marketing
```
SKILLS SECTION:
Channels: [SEO/SEM, Paid Social, Email, Content, Events]
Tools: HubSpot, Salesforce, Google Analytics, Meta Ads Manager
Analytics: UTM tracking, Attribution modelling, A/B testing

EXPERIENCE BULLETS FOCUS:
- Pipeline / revenue generated ($)
- CAC, ROAS, conversion rate improvements
- Campaign scale (reach, impressions, spend managed)
- Content output (pieces, traffic growth)
- Team/agency managed
```

### Operations / Business Analyst
```
SKILLS SECTION:
Analysis: Excel/Sheets, SQL, Python (basic), Tableau
Process: Lean, Six Sigma, BPMN, process mapping
ERP/Tools: SAP, Salesforce, Monday.com, Notion

EXPERIENCE BULLETS FOCUS:
- Cost savings ($, %)
- Process efficiency (time saved, error reduction)
- Scale of operations managed
- Cross-functional projects led
- Reporting/dashboards built
```

### Design (UX/UI)
```
SKILLS SECTION:
Tools: Figma, Sketch, Adobe XD, Framer, Zeplin, Miro
Methods: Design Thinking, User Research, Wireframing, Prototyping
Collaboration: Dev handoff, design systems, accessibility (WCAG)

EXPERIENCE BULLETS FOCUS:
- Products/features designed (user count)
- Usability metrics (task success rate, error rate)
- Research methods used (n= users interviewed)
- Design system contributions (components built)
- Business outcomes tied to design decisions
```

### Fresher / New Grad
```
STRUCTURE ORDER:
1. Education (top, with GPA/rank if strong)
2. Skills
3. Projects (3–5, treated as "experience")
4. Internships (if any)
5. Certifications / Courses
6. Extra-curricular (only if relevant)

PROJECT BULLET FORMAT:
"Built [project name] — a [brief description] using [tech stack].
[Key outcome or metric or learning]."
```

## 4. Seniority Calibration

| Level | Bullet Focus | Resume Length | Summary Tone |
|---|---|---|---|
| Intern / Fresher | Learning, contribution, tools used | 1 page | "Eager, fast learner, [skill] background" |
| Junior (0–2 yrs) | Features built, problems solved | 1 page | "Growing [role] with X foundation" |
| Mid (2–5 yrs) | Outcomes, ownership, scale | 1–1.5 pages | "Experienced [role] specialising in X" |
| Senior (5–8 yrs) | Impact, leadership, systems | 1.5–2 pages | "Senior [role] with track record in X" |
| Staff / Lead (8+) | Org impact, strategy, people | 2 pages | "Technical leader / strategist with X" |
| Director+ | Org building, P&L, vision | 2 pages | "Executive with X-year track record" |

## 5. Weak → Strong Rewrites

| Weak | Strong |
|---|---|
| "Responsible for managing the team" | "Led 5-person engineering team, shipping 3 major features per quarter" |
| "Worked on improving performance" | "Reduced page load time by 62% through lazy loading and CDN optimisation" |
| "Helped with data analysis" | "Analysed 500K customer records to surface churn signals, informing retention campaign that reduced churn by 8%" |
| "Good communication skills" | *(Remove — show it through scope of stakeholders managed)* |
| "Fast learner" | *(Remove — show it by listing tech acquired and deployed)* |
| "Team player" | *(Remove — show it by citing cross-functional projects)* |
| "Managed social media" | "Grew LinkedIn following from 2K to 18K in 6 months through consistent editorial cadence and paid amplification" |
| "Created reports" | "Built executive dashboard in Tableau used weekly by C-suite, consolidating 7 data sources" |

## 6. Gap Handling (Honest Reframing)

### Skills Gap (candidate has related but not exact skill)
```
JD wants: "Kubernetes"
Candidate has: Docker + basic container orchestration

REFRAME: "Containerisation & Orchestration: Docker (production), 
Kubernetes fundamentals (self-study, deploying side projects)"

DON'T: List "Kubernetes" without qualification
DO: Show trajectory toward the skill
```

### Experience Gap (candidate has fewer years)
```
JD wants: 5 yrs | Candidate has: 3 yrs

REFRAME: Lead with impact > years.
Summary: "3 years of [role] with [specific senior-level outcome]"
— Let scope speak louder than tenure
```

### Employment Gap
```
Gap in resume → address proactively if >6 months:

Options (use whichever is true):
- "Career break for [reason]" (caregiving, health, travel — all valid)
- "Independent consulting / freelance work" (if applicable)
- "Upskilling: [specific course/cert completed during gap]"
- "Startup exploration: [project or venture, even if it didn't work]"
```

### Industry Switch
```
Framing strategy:
1. Summary: lead with transferable outcome, not industry
2. Skills: emphasise cross-industry tools (SQL, Python, Excel, Jira)
3. Experience: rewrite bullets to surface the skill, not the industry context
4. Projects: add a relevant portfolio project in target industry
```
