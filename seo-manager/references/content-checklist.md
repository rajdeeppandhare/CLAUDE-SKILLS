# Content Quality & Keyword Optimisation Checklist

## Keyword Placement Checklist

| Location | Priority | Notes |
|---|---|---|
| Page title tag | 🔴 Critical | As close to start as possible |
| H1 heading | 🔴 Critical | Exact or close match |
| First 100 words | 🔴 Critical | Natural usage |
| Meta description | 🟠 High | Once, naturally |
| URL slug | 🟠 High | Short, hyphenated |
| H2/H3 subheadings | 🟡 Medium | Use variants/related terms |
| Image alt text | 🟡 Medium | Descriptive + keyword where natural |
| Last 100 words | 🟢 Low | Reinforces relevance |
| Internal link anchor text | 🟢 Low | Keyword-rich where appropriate |

## Content Depth Checklist

- [ ] Fully answers the search query — nothing left unanswered
- [ ] Longer than competing pages for same keyword (check top 3 results)
- [ ] Includes examples, data, or original insights
- [ ] Has clear sections with descriptive H2/H3 headings
- [ ] Includes images, diagrams, or video where helpful
- [ ] Contains a FAQ section for common related questions
- [ ] Updated within last 12 months (or evergreen and marked as such)
- [ ] Has a clear conclusion or summary

## Readability Checklist

- [ ] Sentences under 20 words on average
- [ ] Paragraphs 2-3 sentences max
- [ ] Uses bullet points and numbered lists
- [ ] No jargon without explanation
- [ ] Active voice preferred over passive
- [ ] Flesch Reading Ease score above 60 for general audiences
- [ ] Uses transition words (however, therefore, additionally)
- [ ] Has a compelling intro that hooks within first 2 sentences

## E-E-A-T Signals (Experience, Expertise, Authority, Trust)

### Experience
- [ ] First-hand experience mentioned or implied
- [ ] Real examples, case studies, or results included
- [ ] Author has relevant credentials shown

### Expertise
- [ ] Content is accurate and well-researched
- [ ] Sources cited where appropriate
- [ ] Technical depth matches audience level

### Authoritativeness
- [ ] Author bio with credentials present
- [ ] Published/updated date visible
- [ ] Referenced by or links to authoritative sources

### Trustworthiness
- [ ] No misleading claims
- [ ] Privacy policy and contact info accessible
- [ ] HTTPS enabled
- [ ] Reviews/testimonials present (for commercial pages)

## Search Intent Matching

| Intent Type | Signs | Content Should Be |
|---|---|---|
| Informational | "how to", "what is", "guide" | Educational, detailed, unbiased |
| Navigational | Brand/site name | Direct, fast to find what they want |
| Commercial | "best", "review", "vs" | Comparative, honest, helpful |
| Transactional | "buy", "price", "order" | Clear CTA, trust signals, pricing |

## Internal Linking Best Practices

- Link to 3-5 relevant internal pages per article
- Use descriptive anchor text (not "click here")
- Link from high-authority pages to new/important pages
- Every page should have at least 1 internal link pointing TO it
- Avoid orphan pages (pages with no internal links)

## Content Freshness

- News/trending topics: update within days
- How-to guides: review every 6-12 months
- Statistics/data posts: update whenever data changes
- Evergreen content: review annually, add "Last updated" date
