# CRO Patterns & Playbooks

Proven patterns for common conversion scenarios. Each includes the problem, the pattern, and example implementations.

---

## Pattern 1: The Leaky Hero

**Problem:** High bounce rate, visitors leave without scrolling.
**Root cause:** Headline fails the 5-second test.

**Fix pattern:**
1. Rewrite headline to answer: what it is + who for + core outcome
2. Add a supporting subheadline with 1–2 specifics
3. Place CTA above the fold, right of or below the headline
4. Add one proof point above the fold (logo bar, stat, or single testimonial)

**Example:**
- Before: "Collaborate better with your team"
- After: "The project tool marketing teams actually ship with — free to start"

## Pattern 2: The Trust Gap

**Problem:** Visitors reach the CTA section but don't click. Heatmaps show attention at the CTA but low clicks.
**Root cause:** Anxiety. The visitor wants to but doesn't trust it yet.

**Fix pattern:**
1. Add social proof directly adjacent to the CTA button
2. Add risk-reversal microcopy below the button ("No credit card required" / "Cancel anytime")
3. Add a trust badge (security, review rating, or money-back guarantee)
4. If B2B: add 3–5 recognisable customer logos near the CTA

---

## Pattern 3: The Form Wall

**Problem:** High form view rate, low form submission rate.
**Root cause:** Too many fields, or fields feel intrusive for the value being offered.

**Fix pattern:**
1. Remove every non-essential field. For lead gen: email only. For trial signup: email + password only.
2. Rewrite the form headline to focus on what the user *gets*, not what they're doing ("Get instant access" vs "Sign up")
3. Add a benefit bullet list beside or above the form
4. Add microcopy: "Free forever. No credit card."
5. Change submit button from "Submit" to "[Action] [What They Get]"

---

## Pattern 4: The Underpowered Landing Page

**Problem:** Paid traffic converting at < 3%.
**Root cause:** Message mismatch, or page doesn't make the full case.

**Fix pattern:**
1. Check message match — headline must mirror ad language
2. Remove global navigation (dedicated landing pages should have no nav)
3. Ensure the page answers all five questions: What? For who? How does it work? Why should I trust you? What do I do next?
4. Add a FAQ section addressing the top 3–5 objections
5. Repeat the CTA at the bottom of the page

---

## Pattern 5: The Unclear Pricing Page

**Problem:** High pricing page views, low plan selection or trial starts.
**Root cause:** Visitors don't know which plan is right for them.

**Fix pattern:**
1. Add "Most Popular" or "Best for [audience]" label to recommended plan
2. Add a plan selector FAQ: "Not sure which plan? [Most users start with Pro]"
3. Reduce plan count if > 4 (cognitive overload)
4. Ensure each plan has a clear CTA button (not just a description)
5. Add a comparison table for feature-heavy products
6. Add a "Talk to sales" or "Not sure? Chat with us" escape valve for enterprise prospects

---

## Pattern 6: The Mobile Conversion Gap

**Problem:** Desktop converts at 8%, mobile at 1.5%. Significant gap.
**Root cause:** Mobile experience has friction desktop doesn't.

**Fix pattern:**
1. Check CTA visibility on mobile (must be above the fold)
2. Test tap target sizes (min 44px height for buttons)
3. Simplify forms for mobile keyboard (avoid dropdowns where possible)
4. Check page speed on mobile (Google PageSpeed Insights)
5. Test pop-up behaviour on mobile (must be easily dismissible)
6. Use sticky CTA bar on mobile for long pages

---

## Pattern 7: The Motivation Gap

**Problem:** Good traffic, reasonable trust signals, but still low conversion. Visitors browse, leave, and don't come back.
**Root cause:** Value proposition isn't compelling enough, or urgency is absent.

**Fix pattern:**
1. Sharpen the headline to focus on outcome transformation, not feature description
2. Add a "before/after" section showing the difference with and without the product
3. Quantify the outcome wherever possible ("saves 5 hours a week" vs "saves time")
4. Add real urgency if available (trial ends, limited spots, cohort starts)
5. Add a secondary CTA for not-ready visitors (newsletter, free resource, demo)
6. Test loss-framing in headline: "Stop losing 6 hours a week to manual reporting"

---

## Pattern 8: The Long-Form Landing Page

**When to use:** Cold traffic that doesn't know your product, high-ticket offers, complex B2B tools.

**Section order:**
1. Hero (headline + subhead + CTA + proof)
2. Problem agitation (name the pain, make it real)
3. Solution introduction (here's a better way)
4. How it works (3–4 steps)
5. Benefits deep-dive (3–5 core benefits with copy)
6. Social proof (testimonials, logos, case study)
7. Objection handling / FAQ
8. Final CTA (value recap + risk reversal + button)

## Microcopy Patterns

Small text that has outsized impact on conversion:

| Location | Weak | Strong |
|---|---|---|
| Below CTA button | (nothing) | "No credit card required. Cancel anytime." |
| Above email field | "Enter your email" | "Get instant access →" |
| Near phone field | "Phone number" | "For your free onboarding call (optional)" |
| Plan CTA | "Get started" | "Start free — upgrade when you're ready" |
| Form headline | "Sign Up" | "Start your free trial" |
| After testimonial | (nothing) | "Join 12,000+ teams already using [Product]" |
